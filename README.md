# math-quiz：四则运算自测系统

## 如何构建

### 依赖

- [Node.js 18 LTS](https://nodejs.org)
- [pnpm 7+](https://pnpm.io/)

### 构建过程

1. 在当前目录打开终端
2. 执行 `pnpm install` 获取依赖包
3. 执行 `pnpm build` 生成站点文件或通过 `pnpm dev` 启动开发服务器
